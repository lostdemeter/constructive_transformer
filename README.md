# The Constructive Transformer

Building a Language Model by Hand — No Gradient Descent, Byte-Exact Arithmetic.

This project demonstrates that a transformer language model can be **constructed by hand** with every weight placed explicitly. The model operates on a 4-state per-axis alphabet with exact integer arithmetic, achieving byte-exact accuracy on concept arithmetic and subject-verb agreement without any training.

## The Claim

> **A transformer LLM is a finite-state machine over a small number of orthogonal axis blocks, with attention routing content across positions and embeddings committing content at positions. The "intelligence" of an LLM, insofar as it lies in concept arithmetic and grammatical agreement, is structurally inevitable given this substrate. It can be constructed by hand, byte-exactly, without any training procedure.**

| **T4-OLMo2** | Phenomenology matches production LLMs | 3/4 holographic findings reproduced |
